# valenitna-reyes
1. Uso de Cookies

Las cookies son pequeños archivos que una página web guarda en el navegador para recordar información del usuario, como sus preferencias, idioma o datos de inicio de sesión. Esto permite ofrecer una experiencia más rápida y personalizada.

2. Creación de Sesiones

Las sesiones son un mecanismo que permite mantener a un usuario identificado después de iniciar sesión. Mientras la sesión esté activa, el usuario puede navegar por la aplicación sin tener que volver a ingresar sus credenciales.

3. Eliminación de Sesiones

La eliminación de una sesión consiste en cerrar el acceso del usuario a la aplicación. Al hacer logout, se eliminan los datos temporales de la sesión, lo que ayuda a proteger la seguridad y privacidad de la cuenta.
